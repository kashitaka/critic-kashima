かしまかずお

年間約40件の美術評論執筆をはじめ、絵画公募団体の外部招聘審査委員や作品講評会に出講など、執筆外活動も旺盛に行う。 略歴 ・1951年 鹿児島県出身 ・1975年 中央大学文学部仏文科卒 ・新宿区中央図書館長、新宿文化センター館長を歴任
